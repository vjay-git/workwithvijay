# workwithvijay.com

Personal freelancing website for Vijay, a Full Stack Developer.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- SEO-first architecture

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
